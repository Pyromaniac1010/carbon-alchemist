# Carbon | The Alchemist

Next.js app (Vercel-ready) that converts feelings + medium into creative prompts using Gemini AI and saves drafts to Firebase Firestore.

## Setup

1. Create project folder and paste files according to the structure.
2. Add handwriting fonts into `public/fonts/`:
   - `OceanTrace-Regular.woff2`
   - `NCL-Gasdrifo-Regular.woff2`
   (Or adjust `globals.css` to use Google Fonts as fallback.)

3. Install dependencies:
```bash
npm install
